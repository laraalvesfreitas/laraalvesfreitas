## Projetos em Destaque

### 🔹 Swag Labs - Testes End-to-End
Projeto de automação de testes End-to-End (E2E) desenvolvido com Cypress utilizando a aplicação Swag Labs como sistema sob teste.

- Automação dos fluxos de Login, Produtos, Carrinho e Checkout;
- Validação da ordenação de produtos;
- Testes de Logout e navegação pelo menu;
- Integração contínua (CI) com GitHub Actions;
- Aplicação de boas práticas de automação de testes.

**Tecnologias:** Cypress, JavaScript, Node.js, GitHub Actions, Git e GitHub.

> 🔗 [Acessar projeto](https://github.com/laraalvesfreitas/cypress-saucedemo-tests)

---

### 🔹 Testes de API - Swagger Petstore
Projeto de testes funcionais de API REST utilizando a Swagger Petstore, com foco no endpoint `/pet`.

- Execução do fluxo completo de CRUD (POST, GET, PUT e DELETE);
- Validação de status codes, payloads e tipos de dados;
- Testes positivos, negativos e exploratórios;
- Identificação e documentação de bugs;
- Organização de evidências e casos de teste.

**Tecnologias:** Postman, Swagger Petstore e API Testing.

> 🔗 [Acessar projeto](https://github.com/laraalvesfreitas/qa-api-tests-swagger-petstore)

---

### 🔹 Casos de Teste BDD - Tela de Login
Projeto de testes manuais utilizando a abordagem BDD (Behavior-Driven Development) para uma tela de login desenvolvida no Figma.

- Elaboração de cenários em Gherkin;
- Testes positivos e negativos;
- Validação de campos obrigatórios;
- Cenários de acessibilidade, usabilidade e segurança;
- Documentação de testes manuais.

**Tecnologias:** Gherkin, Git, GitHub e Análise de Requisitos.

> 🔗 [Acessar projeto](https://github.com/laraalvesfreitas/login-page-test-cases-bdd)

---

### 🔹 Teste Exploratório - Swag Labs
Projeto de testes exploratórios e funcionais realizado na aplicação Swag Labs.

- Identificação de falhas e comportamentos inesperados;

**Tecnologias:** Teste Manual, Markdown e Documentação de Testes.
